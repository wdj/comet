Configure and compile process:

1. Create directory and clone or copy repo into it:
mkdir comet_work
git clone https://code.ornl.gov/wjd/genomics_gpu.git
git checkout <devel,tcb1,etc>

2. Set directory locations:
export COMET_REPO_DIR=<path to genomics_gpu directory>
export COMET_BUILDS_DIR=<path to comet builds directory>
export COMET_INSTALLS_DIR=<path to comet installs directory>

3. Configure CoMet and compile libraries:
genomics_gpu/scripts/configure_all.sh
CoMet sets correct platform by default using hostname

4. Compile CoMet:
Make - genomics_gpu/scripts/make_all.sh
Quick Debug Make - genomics_gpu/scripts/make_all.sh --nompi --norelease --nosingle
Quick Release Make - genomics_gpu/scripts/make_all.sh --nompi --nodebug --nosingle
Quick Debug and Release Make - genomics_gpu/scripts/make_all.sh --nompi --nosingle
View compile script in run_data

4. Run:
View run script in run_data

Compile changes:
CUDA_NVCC_FLAGS:STRING=-I/home/9pe/cutlass/include -I/home/9pe/cutlass/tools/util/include
Remove -Werror from CMAKE_CXX_FLAGS since Cutlass has some strict-aliasing warnings in bfloat16.h and tfloat32.h
For Turing GPUs add: CUDA_NVCC_FLAGS:STRING=-gencode arch=compute_75,code=compute_75
For Ampere GPUs add: CUDA_NVCC_FLAGS:STRING=-gencode arch=compute_80,code=compute_80
