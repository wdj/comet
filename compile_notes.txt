Configure and compile process:

1. Initial settings:
Set COMET_SRC to location of CoMet source code - export COMET_SRC=/home/username/genomics_gpu
GPUSYS2 uses gcc 8.3 downloaded using Spack
Create compile directory - mkdir comet_work

2. Configure CoMet and compile libraries:
${COMET_SRC}/scripts/configure_all.sh COMET_PLATFORM=GPUSYS2

3. Compile CoMet:
Make - ${COMET_SRC}/scripts/make_all.sh
Quick Make - ${COMET_SRC}/scripts/make_all.sh--nompi --norelease --nosingle
View compile scripts in run_data

4. Run:
View run scripts in run_data
